# ALXprodev-advanced_git

This project demonstrates advanced Git techniques and workflows using the GitFlow branching model.

## 📚 Topics Covered

- GitFlow initialization and usage
- Branching strategies (`feature/*`, `release/*`, `hotfix/*`)
- Merging vs Rebasing
- Git stash and cherry-pick
- Collaborative workflows with pull requests
- Git hooks and automation

## 🚀 Purpose

To apply and demonstrate best practices in version control for efficient and collaborative software development.

## 🔧 Tools

- Git
- GitFlow
- GitHub
